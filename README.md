# AI-Powered Document Structuring & Data Extraction

This project converts unstructured PDF text into structured Excel format.

## Features
✔ Extract text from PDF  
✔ Identify Key : Value pairs  
✔ Add contextual comments  
✔ Export to Excel  
✔ Web demo using Streamlit  

## How to Run Locally
pip install -r requirements.txt  
streamlit run app.py

## Output
The tool generates Output.xlsx containing 3 columns:
- Key
- Value
- Comments
